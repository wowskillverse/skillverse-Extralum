Demo 2: fondo SVG fijo

Este ejemplo usa solo scroll vertical. El SVG permanece como fondo constante durante toda la página y sus capas se desplazan según el progreso global del body.

Progreso de scroll 0 → 100%

Al inicio del documento el efecto está en su estado inicial. A medida que bajas, cada capa avanza a distinta velocidad. El punto final se alcanza al llegar al final del body.

Listo para WordPress

Puedes usar este patrón en un bloque HTML o shortcode, manteniendo el SVG y este script encolados desde tu tema o plugin para no depender del editor.

Fin de la página

Aquí el efecto llega a su estado final, porque el progreso de scroll ya es 1.0 (final del body).
